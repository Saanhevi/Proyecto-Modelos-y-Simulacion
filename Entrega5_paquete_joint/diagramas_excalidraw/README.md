# Diagramas Excalidraw (editables)

Convertidos desde `DIAGRAMAS_HIBRIDO.md` (Mermaid) a formato **Excalidraw**.

## Cómo abrirlos / editarlos

1. Entra a https://excalidraw.com
2. Menú **Abrir** (o arrastra el archivo `.excalidraw` a la página)
3. Edita formas, textos y flechas a gusto; **Archivo → Guardar**

También: extensión **Excalidraw** en VS Code / Cursor → clic derecho → Open with Excalidraw.
